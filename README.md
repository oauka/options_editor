# 상품 옵션 에디터 (Product Options Editor)

상품 상세페이지용 옵션 이미지를 쉽게 만들 수 있는 웹 기반 에디터입니다.

## 주요 기능

- 📝 **타이틀 커스터마이징**: 배경색, 텍스트 색상, 악센트 색상 조정
- 🎨 **옵션 관리**: 여러 개의 옵션 추가/삭제/순서 변경
- 🖼️ **이미지 업로드**: 각 옵션별 이미지 업로드 및 위치/크기 조정 (드래그 가능)
- ✏️ **텍스트 스타일링**: 번호, 제목에 굵게/이탤릭/테두리 효과 적용
- ⭕ **원형 오버레이**: 이미지 위에 원형 오버레이 추가 (여러 개 가능)
- 💬 **텍스트 박스**: 추가 텍스트 박스 삽입 (여러 개 가능)
- 📋 **스펙 정보**: 각 옵션에 스펙 정보 추가
- 🎭 **폰트 업로드**: 잘난체, 임팩트 폰트 업로드 지원
- 👁️ **미리보기 모드**: 편집/미리보기 모드 전환
- 💾 **이미지 내보내기**: Canvas로 이미지 생성 및 다운로드

## 설치 방법

### 필수 요구사항
- Node.js 18.0 이상
- npm 또는 yarn

### 설치

```bash
# 저장소 클론
git clone https://github.com/your-username/product-options-editor.git

# 디렉토리 이동
cd product-options-editor

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

개발 서버가 실행되면 `http://localhost:5173`에서 접속할 수 있습니다.

## 빌드

```bash
# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

## 사용 방법

1. **타이틀 설정**: 상단에서 타이틀 텍스트와 색상을 설정합니다. `_`를 사용하여 줄바꿈이 가능합니다.
2. **옵션 추가**: "옵션 추가" 버튼으로 새로운 옵션을 추가합니다.
3. **이미지 업로드**: 각 옵션에 이미지를 업로드하고 드래그하여 위치를 조정합니다.
4. **텍스트 편집**: 번호, 제목, 스펙 정보를 입력하고 스타일을 적용합니다.
5. **미리보기**: "미리보기" 버튼을 클릭하여 최종 결과를 확인합니다.
6. **저장**: "이미지로 저장" 버튼으로 PNG 파일로 다운로드합니다.

## 기술 스택

- **React 18**: UI 프레임워크
- **Vite**: 빌드 도구
- **Tailwind CSS**: 스타일링
- **Lucide React**: 아이콘
- **HTML Canvas API**: 이미지 생성
