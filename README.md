# 상품 옵션 에디터 (Product Options Editor)

상품 상세페이지용 옵션 이미지를 간편하게 제작할 수 있는 React 기반 비주얼 에디터입니다.

## 주요 기능

### 🎨 디자인 커스터마이징
- **타이틀 섹션**: 배경색, 텍스트 색상, 강조 바 색상 설정
- **전체 배경색**: 전체 레이아웃의 배경색 조정
- **폰트 업로드**: 잘난체, Impact 등 커스텀 폰트 지원

### 📦 옵션 관리
- **최대 6개의 옵션** 추가 가능
- **옵션별 설정**:
  - 박스 높이 조절 (300px ~ 800px)
  - 옵션 번호 커스터마이징 (색상, 크기, 굵기, 기울임, 테두리)
  - 옵션명 스타일링 (색상, 크기, 정렬, 굵기, 기울임, 테두리)
  
### 🖼️ 이미지 편집
- **메인 이미지**: 업로드 및 드래그로 위치 조정
- **크기 조절**: 50% ~ 400% 스케일 조정
- **정밀 조정**: 슬라이더 및 수치 직접 입력

### ⭕ 원형 디테일샷 오버레이
- 원형 이미지 추가 및 위치 조정
- 크기 조절 (60px ~ 300px)
- z-index 설정 (메인 이미지 앞/뒤)
- 배경색 커스터마이징
- 원형 내부 이미지 위치 및 크기 조정

### 📝 텍스트 추가
- 자유로운 텍스트 배치
- 글자 크기, 색상, 스타일 설정
- 굵기, 기울임, 테두리 효과

### 📋 옵션 사양
- 최대 10개의 사양 항목 추가
- 텍스트 정렬 (좌/중/우)
- 폰트 크기 및 색상 조절
- 자동 높이 조정

### 💾 내보내기
- **미리보기 모드**: 실시간 결과 확인
- **HTML 내보내기**: 완성된 레이아웃을 HTML 파일로 저장
- **이미지 내보내기**: PNG 형식으로 다운로드

## 기술 스택

- React 18
- Lucide React (아이콘)
- Tailwind CSS (스타일링)
- HTML2Canvas (이미지 변환)

## 설치 및 실행

### 필요 조건
- Node.js 16.x 이상
- npm 또는 yarn

### 로컬 개발

```bash
# 저장소 클론
git clone https://github.com/yourusername/product-options-editor.git

# 디렉토리 이동
cd product-options-editor

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 브라우저에서 http://localhost:3000 접속
```

### Vercel에 배포하기

#### 방법 1: Vercel CLI 사용
```bash
# Vercel CLI 설치 (처음 한번만)
npm install -g vercel

# 배포
vercel

# 프로덕션 배포
vercel --prod
```

#### 방법 2: GitHub 연동
1. GitHub에 코드 푸시
2. [Vercel](https://vercel.com) 접속 및 로그인
3. "New Project" 클릭
4. GitHub 저장소 선택
5. "Deploy" 클릭 - 자동으로 빌드 및 배포됩니다!

### 프로덕션 빌드
```bash
# 로컬에서 빌드
npm run build
```

## 사용 방법

1. **타이틀 설정**: 상단에서 전체 배경색과 타이틀 색상 설정
2. **옵션 추가**: "새 옵션 추가" 버튼으로 옵션 박스 추가 (최대 6개)
3. **이미지 업로드**: 각 옵션에 이미지 업로드
4. **위치 조정**: 이미지를 드래그하거나 슬라이더로 위치 조정
5. **원형 오버레이**: 필요시 원형 디테일샷 추가
6. **텍스트 추가**: 커스텀 텍스트 삽입
7. **사양 추가**: 옵션 사양 정보 입력
8. **미리보기**: "미리보기" 버튼으로 최종 결과 확인
9. **내보내기**: HTML 또는 이미지로 저장
